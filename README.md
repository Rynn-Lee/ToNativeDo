## 🤔 What is this project?
This is a Todo list as you might've guessed<br>
Just wanted to check styling ability of React Native<br>
You can create "Cards" and then add todos in there<br>
Cards are just lists of checkboxes (todos)<br>

## 💻 What tech stack is used?
Front-end: ReactNative, React<br>
State management: Async-Storage<br>
